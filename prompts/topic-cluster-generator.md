# TOPIC CLUSTER GENERATOR PROMPT

Generate 5 supporting blogs (1000–1500 words each) for the main SEO topic.  
Each must:

- Support the pillar blog  
- Target long-tail keywords  
- Include internal links  
- Include metadata  
- Cover actionable insights  

Structure:
- H1 title
- Meta title
- Meta description
- H2 subtopics
- FAQ
- CTA linking to the business service
